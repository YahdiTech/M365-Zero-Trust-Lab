# M365-Zero-Trust-Lab
Enterprise Security Lab: Identity Governance, Endpoint Management, and Data Protection

Implemented Granular Access Control by breaking permission inheritance at the folder level. While the 'HR' site is accessible to staff, the 'Confidential-Payroll' directory is restricted to authorized personnel only, preventing internal data leaks
Validated the Zero-Trust 'Assume Breach' model by performing a permissions audit. Verified that non-authorized internal identities are successfully gated from sensitive HR directories, ensuring data confidentiality.
